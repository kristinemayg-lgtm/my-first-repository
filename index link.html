Step 1: Setting Up Your HTML Document

Document Type Declaration
HTML
Copy
<!DOCTYPE html> <html lang="en">
​
<!DOCTYPE html>
: Defines the document as an HTML5 document.
<html lang="en">
: The root element of the HTML document, specifying the language as English.
Head Section
HTML
Copy
<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Webpage</title> </head>
​
<head>
: Contains meta-information about the document.
<meta charset="UTF-8">
: Sets the character encoding to UTF-8, supporting various characters.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
: Ensures the webpage is responsive on all devices.
<title>
: Sets the title of the webpage, which appears on the browser tab.

Step 2: Adding Content to the Body

Body Section and Main Heading
HTML
Copy
<body> <h1>Welcome to Web Development!</h1>
​
<body>
: Contains the visible content of the webpage.
<h1>
: Defines the main heading of the page, typically the largest and most important text.
Adding a Paragraph
HTML
Copy
<p>This is my first webpage using HTML.</p>
​
<p>
: Defines a paragraph of text, used for blocks of text.
Adding a Subheading and Link
HTML
Copy
<h2>Learn More About Web Development</h2> <p>Visit the <a href="<https://developer.mozilla.org/en-US/docs/Learn>">MDN Web Docs</a> to learn more.</p>
​
<h2>
: Defines a subheading, which is smaller than
<h1>
but still important for structuring content.
<a href="URL">
: Creates a hyperlink to another webpage. The text between the tags is clickable.
Adding an Image
HTML
Copy
<h2>Here’s a Cool Image</h2> <img src="<https://www.example.com/image.jpg>" alt="A cool image">
​
<img src="URL" alt="description">
: Embeds an image into the webpage. The
src
attribute specifies the image path, and
alt
provides alternative text for accessibility.

Step 3: Adding Lists and Tables

Adding an Ordered List
HTML
Copy
<h2>Topics I'm Interested In</h2> <ol> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ol>
​
<ol>
: Defines an ordered list, which is numbered.
<li>
: Represents each item in the list.
Adding a Table
HTML
Copy
<h2>My Favorite Programming Languages</h2> <table border="1"> <tr> <th>Language</th> <th>Experience Level</th> </tr> <tr> <td>HTML</td> <td>Intermediate</td> </tr> <tr> <td>CSS</td> <td>Beginner</td> </tr> <tr> <td>JavaScript</td> <td>Beginner</td> </tr> </table>
​
<table>
: Defines a table.
<tr>
: Defines a row in the table.
<th>
: Defines a header cell, usually bold and centered.
<td>
: Defines a standard data cell in the table.

Step 4: Finalizing the HTML Document

Closing the Body and HTML Tags
HTML
Copy
</body> </html>
​
</body>
: Closes the body section.
</html>
: Closes the HTML document
Testing Your Web Page
Save your
index.html
file.
Open it in your web browser to see the complete webpage.
